Vision Transformer (ViT)

Google's ViT model splits images into 16×16 pixel patches and uses a Transformer architecture — the same technology behind GPT — to classify images into 1000 categories.

Hugging Face API

We use the free Hugging Face Inference API to run the model. No GPU needed — just send an HTTP request with your image and get back predictions with confidence scores.

How It Works

Upload → Frontend sends image to Python backend → Backend forwards to Hugging Face API → ViT model analyses → Results returned → Displayed with confidence scores.
