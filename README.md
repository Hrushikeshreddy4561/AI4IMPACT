# AI for Impact (AI4IMPACT)

Welcome to the **AI for Impact** repository. This collection contains two distinct AI-driven applications designed to showcase the power of modern machine learning and LLM integrations.

## 🚀 Projects Included

### 1. [AI Image Classifier](./ai-image-classifier)
A Python-based application that uses Hugging Face models to classify images with high accuracy.
- **Technologies**: Python, Streamlit/Flask, Hugging Face Transformers.
- **Key Features**: Instant image recognition, support for multiple categories, and a user-friendly interface.

### 2. [LLM Chatbot with Streaming](./llm-chatbot)
A modern, responsive web application that provides a real-time chat experience using the Google Gemini 1.5 Flash model.
- **Technologies**: Next.js 14, React, Tailwind CSS, Google Generative AI SDK.
- **Key Features**: Real-time streaming responses, conversational memory, and a premium dark-themed UI.

---

## 🛠️ Getting Started

To get started with either project, navigate into the respective directory and follow the instructions in its `README.md` file.

```bash
# To explore the Image Classifier
cd ai-image-classifier

# To explore the LLM Chatbot
cd llm-chatbot
```
